# Rock-Paper-Scissors
A simple Rock Paper Scissors game built using Python where a user plays against the computer through a command-line interface.


**Project Overview**

This project is a Rock Paper Scissors game developed using Python, where a player competes against the computer. The game runs in the command-line interface (CLI) and uses Pythonâ€™s built-in features to generate random moves and determine the winner.
It is a beginner-friendly project designed to help understand basic Python programming concepts such as conditionals, loops, functions, and random number generation.


**Features**

1.User vs Computer gameplay

2.Random move generation by the computer

3.Clear win/lose/tie result display

4.Simple and easy-to-understand logic

5.Beginner-friendly Python code


**Working Logic**

1.The user selects Rock, Paper, or Scissors.

2.The computer randomly selects one option.

3.The program compares both choices.

4.The result (Win / Lose / Tie) is displayed.
